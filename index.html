Bienvenido al Encriptador de Texto

En esta página encontraras la posibilidad de encriptar textos y también de desencriptarlos, sigue las instucciones en cada tipo de encriptador, y recuerda que el encriptador dinámico cambia cada que se actualiza la página, mientras que el encriptador estático sigue igual al paso del tiempo, por lo que puedes entrar a utilizarlo y desencriptar tu información en cualquier momento; por último recuerda solo encriptar textos compuestos solamente por letras minúsculas sin acentos y los espacios.

ENCRIPTADOR

ESTÁTICO

Este encriptador contiene un código de encriptación prediseñado, si quieres conocer el código de encriptación utilizado, ve los resultados de la consola, donde se muestra el vector con los 28 lugares del abecedario contando el espacio y la ñ asignados con tres o cuatro caracteres para cada letra.

Si eres una persona que le gustan los retos, este es un reto lógico muy bueno, puedes generar la enciptación de un texto y tratar de decifrar el código por tí mismo, puedes pasar unos minutos, unas horas o algunos dias, dependiendo de tu lógica al procesar los datos. Pruébalo y reta a tu cerebro a algo diferente presionando el botón debajo.

ENCRIPTADOR

DINÁMICO

Este encriptador genera un nuevo código de encriptación cada vez que se actualiza la página, si quieres conocer el código de encriptación utilizado en cada sesión, ve los resultados de la consola, donde se muestra el vector con los 28 lugares del abecedario (contando como el lugar 27 el espacio y el 28 la ñ) asignados con tres valores en código asci para cada letra. Lo que se puede resumir en que cada letra toma el valor de tres letras aleatorias cada que se actualiza la página del encriptador.

Con este encriptador podrás divertirte un poco o incluso invitar a alguien a intentar decifrarlo con alguna frase hecha por ustedes mismo esto les da mayor oportunidad de indagar y adivinar que código fue utilizado por ustedes mismos. Te invitamos a probarlo haciendo click en el botón debajo.

ENCRIPTADOR

COMPLETO

Este encriptador genera un nuevo código de encriptación cada vez que se actualiza la página, si quieres conocer el código de encriptación utilizado en cada sesión, ve los resultados de la consola, donde se muestra el vector con los 96 lugares de caracteres (contando como el lugar 96 la ñ) asignados con tres valores en código asci para cada letra. Lo que se puede resumir en que cada caracter toma el valor de tres caracteres aleatorios cada que se actualiza la página del encriptador.

Si eres una persona que le gustan los retos, este es el reto mas grande, puedes generar la enciptación de un texto y tratar de decifrar el código por tí mismo, puedes pasar unos minutos, unas horas o algunos dias, dependiendo de tu lógica al procesar los datos. Pruébalo y prueba realizar una doble o triple encriptación para mayor dificultad.
